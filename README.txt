SIMPLEX prototype
=================

!!! Using NPM !!!

Dev: ```monaca preview``` or ```npm run dev```

Build APK: ```monaca build```

Build and transfer to device/emulator: ```npm run mobile```

Configuration
-------------

This app was initially developed to run on the secondary screen (tablet).
The artwork was displayed on the main screen (public display).
With the following setting, the application can fully run on the main screen. Input, in this case, must be given on the main screen with touch.
